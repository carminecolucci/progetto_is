## Build & Run

Le seguenti dipendenze sono necessarie al funzionamento dell'applicativo:
- OpenJDK-22.0.1
- Oracle MySQL Server 8.0.38
- mysql-connector-j-8.4.0.jar

Per eseguire i test:
- junit-4.10.jar
- bluecatcode.junit.4.10.extended.jar
