# Progetto di Ingegneria del Software


# Build

## Dependencies
- Install [MikTex](https://miktex.org/)

- Install [Perl](https://strawberryperl.com/)

## Build documentation
From `docs` folder

To build the project documentation and generate the pdf file, run:
```bash
make build
```

To start a live server, run:
```bash
make run
```
